# Conversion Optimization Frameworks

## AIDA Framework

**Attention вҶ’ Interest вҶ’ Desire вҶ’ Action**

| Stage | Element | Example |
|-------|---------|---------|
| Attention | Hero headline | "Ship 10x faster with AI" |
| Interest | Feature showcase | Show dashboard screenshot |
| Desire | Social proof + benefits | "Used by 500+ teams" |
| Action | Clear CTA | "Start free trial вҶ’" |

## Time-to-Value Optimization

Reduce friction between signup and first value moment:

1. **Minimize form fields** вҖ” Name + email only for signup
2. **Skip email verification** вҖ” Verify later, let them explore now
3. **Template/sample data** вҖ” Pre-populate with example project
4. **Guided first action** вҖ” Tooltip tour pointing to key action
5. **Success celebration** вҖ” Confetti/animation on first milestone

**Target:** Under 2 minutes from signup to "aha moment"

## Friction Reduction Checklist

- [ ] Forms have max 3-4 fields visible at once
- [ ] Primary CTA has contrasting color
- [ ] Error messages appear inline, not as alerts
- [ ] Loading states show progress, not just spinners
- [ ] Mobile buttons are at least 44x44px tap targets
- [ ] No required account for viewing public content
- [ ] Social login available (Google, GitHub)

## CTA Best Practices

**DO:**
- Use action verbs ("Start building", "Get started free")
- Show what happens next ("No credit card required")
- Make primary CTA visually dominant (size + color)
- Repeat CTA at top and bottom of page

**DON'T:**
- Use "Submit" or "Click here"
- Hide CTA below the fold
- Use same color for CTA and regular links
- Add too many competing CTAs

## Pricing Page Patterns

- **3 tiers maximum** вҖ” Too many choices paralyze
- **Highlight recommended plan** вҖ” Border, badge, or scale
- **Annual toggle default** вҖ” Show annual savings prominently
- **Feature comparison** вҖ” Show what each tier includes
- **Enterprise CTA** вҖ” "Contact us" for custom pricing

## Onboarding Flow

```
Step 1: Welcome + Name           (minimal friction)
Step 2: Choose use case          (personalize experience)
Step 3: Create first project     (immediate value)
Step 4: Invite team member       (social lock-in, optional)
Step 5: Dashboard with guidance  (tooltips highlight key features)
```
